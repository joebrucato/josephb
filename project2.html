Snowbird Snowfall Predictor

About This Project

Created a regression model using scikit-learn to predict seasonal snowfall at Snowbird Ski Resort, achieving a mean absolute error of 17.74 inches and a prediction accuracy within 0.45 inches for the 2023-2024 season. The project leveraged pandas for data manipulation, matplotlib for visualizing trends, and Jupyter Notebooks for an interactive development process. Inspired by my upcoming ski trip to Snowbird, this project allowed me to analyze potential snow conditions and share the insights with my fellow travelers, helping us better prepare and set expectations for the trip.
